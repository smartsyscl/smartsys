# SmartSYS Digital Solutions

Este proyecto es una aplicación web construida con Next.js de SmartSYS Digital Solutions. Su objetivo principal es mostrar los servicios digitales ofrecidos, con un enfoque particular en la automatización y soluciones empresariales. La aplicación cuenta con un catálogo de servicios detallado, una sección de testimonios de clientes satisfechos, un formulario de contacto para facilitar la comunicación y un portafolio que exhibe proyectos exitosos.

La aplicación está optimizada para motores de búsqueda (SEO) y, como característica destacada, incluye una herramienta de recomendación de servicios impulsada por inteligencia artificial que sugiere servicios relevantes según las necesidades del usuario. El idioma principal de la interfaz es el español, con soporte secundario para el inglés.

El diseño visual sigue un estilo limpio y basado en cuadrícula, utilizando una paleta de colores que incluye azul oscuro (`#3F51B5`), azul muy claro (`#E8EAF6`) y morado suave (`#9575CD`). La tipografía se compone de la fuente 'Poppins' para los encabezados y 'Inter' para el cuerpo del texto, ambas fuentes modernas y limpias.

## Características Principales

- Catálogo de Servicios: Presentación detallada de los servicios digitales ofrecidos.
- Sección de Testimonios: Muestra las experiencias positivas de los clientes.
- Formulario de Contacto: Permite a los usuarios enviar consultas de manera sencilla.
- Portafolio de Proyectos: Exhibe trabajos de automatización exitosos.
- Optimización SEO: Estrategias implementadas para mejorar la visibilidad en buscadores.
- Herramienta de Recomendación de Servicios con IA: Sugiere servicios relevantes basado en las necesidades del negocio del usuario.
- Soporte Multilingüe: Español (principal) e Inglés (secundario).

## Tecnologías Utilizadas

- [Next.js](https://nextjs.org/): Framework de React para aplicaciones web con renderizado del lado del servidor.
- [React](https://reactjs.org/): Biblioteca de JavaScript para construir interfaces de usuario.
- [TypeScript](https://www.typescriptlang.org/): Lenguaje tipado basado en JavaScript.
- [Tailwind CSS](https://tailwindcss.com/): Framework de CSS para un desarrollo rápido y flexible.
- [Firebase](https://firebase.google.com/): Plataforma de Google para el desarrollo de aplicaciones web y móviles, utilizada para la base de datos, autenticación, etc.
